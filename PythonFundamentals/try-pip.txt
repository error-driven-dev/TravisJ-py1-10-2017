- pip install Django : This command installed the Django framework.

- pip list: Shows all installed modules/packages

- pip install Django: This attempted to re-install Django but the operation didn't need to be done.

- pip freeze: Shows all installed modules/packages by requirements/dependencies

- pip uninstall Django : Removed Django framework

- pip show Django : Nothing to show

- pip search Flask : Shows all modules associated with Flask framework along with a brief description of their purpose
